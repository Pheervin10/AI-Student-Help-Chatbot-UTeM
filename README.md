# AI-Based Student Help Chatbot for Academic Support at UTeM

## Project Description
This project focuses on the development and management of an AI-based student help chatbot designed to support academic enquiries at Universiti Teknikal Malaysia Melaka (UTeM). The chatbot assists students by providing instant responses to frequently asked academic questions.

## Objectives
- To provide 24/7 academic support to students
- To reduce manual workload on university staff
- To apply AI project management principles in a real project

## Tools and Technologies
- Python
- GitHub
- Google Colab
- Microsoft Excel (Project Planning)
- Natural Language Processing (NLP)

## Project Management Focus
This project emphasises planning, scheduling, risk management, and execution of an AI-based system using structured project management practices.
